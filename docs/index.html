pongstagr.am

is a jQuery plugin that lets display your instagram media to your website. Basically same plugin minus Bootstrap styles. so that means you can choose what ever modal plugin or front-end framework to use by extending it via your own css or js.

requirements:

For the most part, Instagramâ€™s API only requires the use of a client_id. A client_id simply associates your server, script, or program with a specific application. However, some requests require authentication - specifically requests made on behalf of a user. Authenticated requests require an access_token. These tokens are unique to a user and should be stored securely. Access tokens may expire at any time in the future. more here.


  1. Instagram account, User ID and Access Token
  2. jQuery 1.7.0+

or you can head over here to use the Bootstrap version (one that relies on Bootstrap Front-end).

usage:

  1. User ID: If you have zero idea what your user id is, you may head to this link.

  2. Access Token: to get your access token, you may head to this link,
    make sure you follow the instructions on the "How do I get my client id?" link.

  3. jQuery must be initialised first and so as the bootstrap plugin: 

    <script src='//ajax.googleapis.com/ajax/libs/jquery/1.9.1/jquery.min.js'></script>
<script src='path/to/js/bootstrap.min.js'></script>  
<script src='path/to/js/pongstagr.am.js'></script>

    using the plugin: 

    $('div#selector').pongstgrm({
  accessId     : YourAccessID,
  accessToken  : YourAccessToken
});

    By default the plugin displays eight (8) most recent media.
